# WeatherPSP
a set of programs that use the tls4psp fetcher for weather purposes.

It is very very self explanatory and easy to use. The only thing of note is that the average handshake time is 10 seconds in 1.0 due to inefficiencies within tls4psp.
Thanks to using newer versions of tls4psp, handshake time has dropped to 2 - 4 seconds on 1.5 and above.
